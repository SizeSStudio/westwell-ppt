# Visual Grammar Pass

Before generating any Westwell deck with 5+ pages, establish the deck's
visual grammar with two representative sample slides, render them, and get
confirmation before batch production.

## Why

Westwell decks often fail when the content is correct but the rhythm is not:
too many identical light slides, too many cards, not enough image-led pages,
or data pages without a strong numeric anchor. The visual grammar pass catches
that while only two slides are cheap to revise.

## Required 2-Slide Showcase

Create and preview:

1. **Anchor slide** — cover, `hero()`, chapter, or statement page.
2. **Evidence slide** — `big_numbers()`, `lead_image()`, `pipeline()`,
   `image_grid()`, or a dense analytic/content slide.

The two slides should be visually different enough to set the rules for the
whole deck: dark/light rhythm, title scale, image treatment, metadata style,
and content density.

## Page Four Questions

Answer these for every dummy page:

| Question | Decision it drives |
|---|---|
| Narrative role | hero / data / quote / image-led / dense analysis / summary |
| Audience distance | meeting-room projection vs laptop review vs printed appendix |
| Visual temperature | executive minimal / data editorial / soft systems / impact |
| Capacity estimate | whether content fits one slide or must split |

## Westwell-Compatible Styles

Use these as layout and rhythm references only. Do not import their palettes
over Westwell blue/teal.

| Direction | Use for | Builder choices |
|---|---|---|
| Pentagram / Data Editorial | benchmark, market analysis, data proof | `big_numbers()`, `rowlines()`, table/chart image pages |
| Build / Executive Minimal | board, strategy, conclusion, decision pages | `hero()`, `quote_editorial()`, sparse `text()` |
| Takram / Soft Systems | platform architecture, ecosystem, energy coordination | `pipeline()`, `lead_image()`, `image_grid()`, soft diagrams |

## Rhythm Defaults

- No more than three consecutive standard content slides.
- Insert a hero, quote, big-number, or image-led page every 3-4 pages.
- Use dark slides for chapter pivots, strong conclusions, and executive
  pauses; use light slides for data, diagrams, and screenshots.
- Alternate sparse and dense pages. If a page has 4+ content blocks, the next
  page should breathe.

## Confirmation Criteria

The two preview slides are approved when:

- The title hierarchy is readable at meeting-room distance.
- The Westwell brand still feels primary.
- The evidence slide has one unmistakable visual anchor.
- Metadata and footers add context without competing with the title.
- No placeholder is pretending to be real evidence.
